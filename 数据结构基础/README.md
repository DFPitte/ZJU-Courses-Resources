## 数据结构基础
本课程尽管只有2.5学分，但事情绝对是大一下专业课中最多的。课程主要介绍了一些基础的数据结构，如栈，队列，二叉树和堆等等。课件、作业及考试均为英文。笔者是杨子祺老师班上的，也推荐选该老师的课。但是，本课程由于是课程组统一规定的，并且给分主要看硬性成绩（即期末考），故实际选择哪个老师其实影响并没有想象中的那么大。<br>
当然，还是有必要作出一个好选择的（比如不要选cyll。
还有，在学期初会有一次类似于水平测试的东西，决定你的projects是normal难度还是hard难度。hard的题目可比normal难太多了，并且hard的分值也只能多5分，不如选normal，再做个bonus（一道题有两分，两道题共计四分），平时分一般都会给满的。hard的project2自动微分题目代码量过大，已经显得有些无意义（当然想挑战一下自己的当然也可以选hardhhhhh）,并且实验报告的繁文缛节过多，大家也基本上都是chatgpt生成的（甚至直接生成英文hhhh，互评制度也显的没有意义）。所以这门课也是事情最多的课了。。。
杨子祺老师班上有三次小测（不会提前通知），相当于点名（来了基本都会给满的，但每次占总评中的3分，不来的话还是会有影响）。当然老师的课还是很值得一听的！
